# Telco Customer Churn Analysis

Analisi end-to-end del Customer Churn nel settore telecomunicazioni.
Il progetto include Data Cleaning, Analisi Esplorativa (EDA), Clustering per la segmentazione clienti e modelli di Classificazione (Machine Learning) per prevedere l'abbandono.

## Struttura
- `data/`: Contiene il dataset originale.
- `notebooks/`: Analisi e modelli.

## Obiettivo
Identificare i fattori di rischio e prevedere quali clienti abbandoneranno il servizio.
